# TruthLens# 🔍 TruthLens

TruthLens is an AI-powered fake news and claim verification platform that combines live news search and machine learning to verify the authenticity of news claims.

## 🚀 Features

- Verify news claims
- AI-powered prediction
- Live news verification
- React + Vite frontend
- Flask backend
- Chrome extension

## 🛠 Tech Stack

- React
- Vite
- Python
- Flask
- Scikit-learn
- Render
- Vercel

## 📂 Project Structure

```
backend/
frontend/
extension/
README.md
```

## ▶️ Run Project

### Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
