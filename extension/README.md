# TruthLens Browser Extension

This is a local Chrome/Edge extension that sends highlighted text to the Vite app.

1. Start the TruthLens frontend at `http://127.0.0.1:5173/`.
2. Open `chrome://extensions` or `edge://extensions`.
3. Enable developer mode.
4. Choose "Load unpacked" and select this `extension` folder.
5. Highlight a claim on any page, right-click, and choose "Check with TruthLens".
